# ScriptReview

**Workflow Stage: Script Development Phase - Review & Enhancement**

## Overview

The **ScriptReview** stage represents the editorial review phase where the script draft is analyzed, refined, and enhanced for optimal narrative quality.

## Position in Workflow

```
ScriptDraft → [ScriptReview] → ScriptApproved → Voiceover
```

## Purpose

Strengthen the script's pacing, emotional impact, and narrative quality. This stage focuses on:

- Pacing optimization
- Emotional beat enhancement
- Suspense curve refinement
- Clarity improvements
- Engagement maximization

## Key Activities

1. **Analyze Pacing** - Review rhythm and tempo of content delivery
2. **Enhance Emotional Beats** - Strengthen key emotional moments
3. **Refine Suspense** - Optimize tension and release patterns
4. **Improve Clarity** - Ensure message is clear and compelling
5. **Test Engagement** - Verify content maintains audience interest

## Review Focus Areas

### Narrative Quality
- Story arc effectiveness
- Character development (if applicable)
- Plot coherence and logic
- Thematic consistency
- Message clarity

### Pacing & Rhythm
- Section length balance
- Transition smoothness
- Tempo variation
- Energy level management
- Attention retention

### Emotional Impact
- Emotional beat placement
- Intensity modulation
- Empathy creation
- Tension building
- Satisfaction delivery

### Technical Elements
- Grammar and syntax
- Word choice optimization
- Readability score
- Speech flow naturalness
- Performance readiness

## Review Process

1. **Initial Read-Through** - Experience script as audience would
2. **Structural Analysis** - Evaluate arc, pacing, and flow
3. **Line-by-Line Edit** - Refine language and delivery
4. **Timing Verification** - Confirm length targets
5. **Final Quality Check** - Ensure all standards are met

## Deliverables

- Reviewed script with tracked changes
- Editorial notes and suggestions
- Pacing analysis document
- Emotional beat map
- Revision recommendations

## Quality Checkpoints

Before moving to ScriptApproved:
- [ ] Pacing is optimized for engagement
- [ ] Emotional beats are strong and well-placed
- [ ] Narrative arc is compelling
- [ ] All technical issues are resolved
- [ ] Script reads naturally when spoken
- [ ] Length is within target range
- [ ] Tone is consistent and appropriate

## Revision Types

### Major Revisions
- Structural changes
- Section reordering
- Content additions/deletions
- Arc modifications

### Minor Revisions
- Word choice improvements
- Grammar corrections
- Timing adjustments
- Transition refinements

## Transition Criteria

A script moves from Review stage when:
- ✅ All editorial feedback is addressed
- ✅ Pacing is optimized
- ✅ Emotional beats are enhanced
- ✅ Technical quality is high
- ✅ Approved by editorial review
- ✅ Ready for voiceover production
