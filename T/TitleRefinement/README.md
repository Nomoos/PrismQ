# TitleRefinement

**Align title with final script**

## Purpose

Refine the title to align perfectly with the finalized script content.

## State Position

```
ScriptDraft → [TitleRefinement] → Finalization
                  ↑
          ScriptImprovements (from improvement hub)
```

## Key Activities

- Review title-script alignment
- Refine title wording
- Optimize for content match
- Test title variations
- Finalize title choice

## Deliverables

- Refined title
- Alignment justification
- Title variations tested
- Final title selection

## Quality Gates

- Perfect script alignment
- Clear and compelling
- Appropriate length
- Engagement potential

## Transitions

**Forward:**
- → Finalization (title aligned with script)

**Backward:**
- ← ScriptDraft (from script phase)
- ← ScriptImprovements (from improvement hub)

**Terminate:**
- → Archived (abandoned)
