# GrammarReview

**Check grammar, spelling, punctuation**

## Purpose

Comprehensive grammar, spelling, and punctuation review.

## State Position

```
Editing → [GrammarReview] → ConsistencyCheck
              ↓
       ScriptImprovements (if major issues)
```

## Key Activities

- Grammar checking
- Spelling verification
- Punctuation review
- Syntax validation
- Language correctness

## Deliverables

- Grammar review report
- Error list
- Correction recommendations
- Quality score

## Quality Gates

- Zero grammar errors
- Correct spelling
- Proper punctuation
- Valid syntax

## Transitions

**Forward:**
- → ConsistencyCheck (grammar approved)

**Backward:**
- → ScriptImprovements (major language issues)

**Terminate:**
- → Archived (abandoned)
