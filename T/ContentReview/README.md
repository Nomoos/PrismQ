# ContentReview

**Review content structure and flow**

## Purpose

Review the script's content structure, narrative flow, and overall coherence.

## State Position

```
ScriptDraft → [ContentReview] → Editing
```

## Key Activities

- Review narrative structure
- Assess story flow
- Check pacing
- Evaluate engagement
- Identify structural issues

## Deliverables

- Content review report
- Structural feedback
- Flow assessment
- Revision recommendations

## Quality Gates

- Logical structure
- Smooth flow
- Engaging narrative
- Clear messaging

## Transitions

**Forward:**
- → Editing (structure approved)

**Terminate:**
- → Archived (abandoned)
