# TitleDraft

**Draft initial title**

## Purpose

Create the initial title draft based on the outline and concept.

## State Position

```
Outline → [TitleDraft] → ScriptDraft
```

## Key Activities

- Draft working title
- Test title variations
- Align title with concept
- Consider SEO/engagement
- Set title direction

## Deliverables

- Initial title draft
- Title variations (2-3 options)
- Title justification
- Target audience alignment

## Quality Gates

- Clarity and appeal
- Concept alignment
- Appropriate length
- Memorable/catchy

## Transitions

**Forward:**
- → ScriptDraft (title drafted)

**Terminate:**
- → Archived (abandoned)
