# ConsistencyCheck

**Verify consistency (names, facts, timeline)**

## Purpose

Verify internal consistency of names, facts, timeline, and details throughout the script.

## State Position

```
GrammarReview → [ConsistencyCheck] → ToneCheck
                      ↓
              ScriptImprovements (if inconsistencies)
```

## Key Activities

- Check name consistency
- Verify factual accuracy
- Validate timeline
- Cross-reference details
- Identify contradictions

## Deliverables

- Consistency report
- Inconsistency list
- Fact verification
- Timeline validation

## Quality Gates

- No name mismatches
- Factual accuracy
- Timeline coherence
- Detail consistency

## Transitions

**Forward:**
- → ToneCheck (consistency verified)

**Backward:**
- → ScriptImprovements (inconsistencies found)

**Terminate:**
- → Archived (abandoned)
