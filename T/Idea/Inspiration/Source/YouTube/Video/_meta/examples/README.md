# Video Module Examples

This directory contains usage examples for the YouTube Video scraping module.

## Examples

### analyzed_video_mera_bo_staff.py

**YouTube Video Analysis: Neurodivergent Empowerment Story**

This example demonstrates how to analyze a YouTube video and create an IdeaInspiration 
that can be used in the PrismQ content production pipeline.

**Video Summary:**
A parent shares the story of their 11-year-old autistic daughter Mera, who used 3 years 
of dedicated bo staff training (with her late grandfather's hand-carved staff) to defend 
her best friend Theo (who has ADHD) from a violent bully who broke into their home.

**Key Features Demonstrated:**
- Video subtitle analysis and text extraction
- IdeaInspiration data structure creation
- Transformation to full Idea model for content production
- Story structure analysis (skeleton, beat sheet, emotional arc)
- Target audience and platform identification
- Content scoring and category relevance

**Run the example:**
```bash
python analyzed_video_mera_bo_staff.py
```

**Story Themes:**
- Neurodivergent empowerment
- Friendship and loyalty  
- Dedicated practice/training
- Legacy from loved ones
- Self-defense in protection of others

**Target Platforms:** YouTube, TikTok, Instagram, Podcast
