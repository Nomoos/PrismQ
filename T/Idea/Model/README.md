# PrismQ.Idea.Model

**Content idea data model for the PrismQ universal content generation workflow.**

---

## What is PrismQ.Idea?

The Idea module is the second stage in the PrismQ content workflow:

```
IdeaInspiration ‚Üí Idea ‚Üí Script ‚Üí Proofreading ‚Üí Publishing
```

An **Idea** represents a distilled, actionable concept that can be:
- Created independently OR fused from multiple IdeaInspiration sources
- Released simultaneously as **text**, **audio**, and **video**
- Distributed across multiple platforms (YouTube, Spotify, Medium, etc.)
- Generated by AI with rich contextual fields

---

## Quick Start

```bash
# Install
cd Idea/Model
pip install -e .

# Setup database
./setup_db.sh  # or setup_db.ps1 on Windows

# Create an idea
python -c "
from idea import Idea, ContentGenre
idea = Idea(
    title='My First Idea',
    concept='A great concept',
    target_platforms=['youtube'],
    target_formats=['video'],
    genre=ContentGenre.EDUCATIONAL
)
print(idea)
"
```

## Quick Examples

### Standalone Idea
```python
from idea import Idea, ContentGenre

idea = Idea(
    title="Python Basics",
    concept="Teaching Python fundamentals",
    target_platforms=["youtube", "medium"],
    target_formats=["text", "video"],
    genre=ContentGenre.EDUCATIONAL,
    keywords=["python", "tutorial"],
    outline="1. Setup\n2. Variables\n3. Functions",
    skeleton="Intro ‚Üí Theory ‚Üí Practice"
)
```

### Fused from Inspirations
```python
idea = Idea.from_inspirations(
    inspirations=[insp1, insp2, insp3],
    title="Digital Detective Stories",
    concept="Solving internet mysteries",
    target_platforms=["youtube", "spotify", "medium"],
    target_formats=["text", "audio", "video"],
    genre=ContentGenre.TRUE_CRIME
)
```

### Database Operations
```python
from idea_db import IdeaDatabase

db = IdeaDatabase("idea.db")
db.connect()

# Save
idea_id = db.insert_idea(idea.to_dict())

# Retrieve
idea_dict = db.get_idea(idea_id)
idea = Idea.from_dict(idea_dict)

# Query
drafts = db.get_ideas_by_status("draft")

db.close()
```
