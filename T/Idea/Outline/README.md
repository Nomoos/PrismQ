# Outline

**Create structured outline**

## Purpose

Create a detailed structural outline for the story, defining the flow and key points.

## State Position

```
IdeaReview → [Outline] → TitleDraft
```

## Key Activities

- Define story structure
- Identify key story points
- Establish narrative flow
- Set content boundaries
- Plan story arc

## Deliverables

- Detailed content outline
- Story structure framework
- Key point list (3-6 points)
- Flow diagram

## Quality Gates

- Logical flow
- Clear structure
- Complete coverage
- Appropriate length

## Transitions

**Forward:**
- → TitleDraft (outline complete)

**Terminate:**
- → Archived (abandoned)
