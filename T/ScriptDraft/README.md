# ScriptDraft

**Write initial script**

## Purpose

Write the initial script based on the outline and title draft.

## State Position

```
TitleDraft → [ScriptDraft] → ContentReview
               ↓
         TitleRefinement (optional)
```

## Key Activities

- Write full script
- Follow outline structure
- Develop narrative voice
- Include dialogue/descriptions
- Set initial pacing

## Deliverables

- Complete initial script
- Word count tracking
- Scene/section breakdown
- Draft metadata

## Quality Gates

- Completeness
- Outline adherence
- Basic coherence
- Appropriate length

## Transitions

**Forward:**
- → ContentReview (draft complete)
- → TitleRefinement (title needs work)

**Terminate:**
- → Archived (abandoned)
