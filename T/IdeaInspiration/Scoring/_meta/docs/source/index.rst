PrismQ IdeaInspiration Scoring
==============================

Welcome to the PrismQ IdeaInspiration Scoring documentation. This module provides
a scoring engine for evaluating idea inspirations from various content sources.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   api

Features
--------

* PEP compliant configuration (PEP 517/518/621/660)
* Type checking with MyPy (PEP 484/526/544/561)
* Code quality with Ruff (PEP 8/257)
* Comprehensive logging system
* Test framework setup (pytest)
* Development scripts for Windows and Linux

Target Platform
---------------

* **Operating System**: Windows
* **GPU**: NVIDIA RTX 5090 (32GB VRAM)
* **CPU**: AMD Ryzen processor
* **RAM**: 64GB DDR5

Quick Start
-----------

1. Clone the repository
2. Run ``scripts/setup.sh`` (Linux) or ``scripts\setup.bat`` (Windows) to install dependencies
3. Configure environment using ``.env`` file
4. Start developing your scoring algorithms
