# ToneCheck

**Validate tone and style match**

## Purpose

Validate that the tone and style match the intended genre and audience.

## State Position

```
ConsistencyCheck → [ToneCheck] → ReadabilityReview
                      ↓
              ScriptImprovements (if tone mismatch)
```

## Key Activities

- Assess tone consistency
- Validate style alignment
- Check genre fit
- Evaluate voice
- Review emotional impact

## Deliverables

- Tone analysis report
- Style assessment
- Genre alignment check
- Voice consistency review

## Quality Gates

- Consistent tone
- Appropriate style
- Genre match
- Clear voice

## Transitions

**Forward:**
- → ReadabilityReview (tone approved)

**Backward:**
- → ScriptImprovements (tone mismatch)

**Terminate:**
- → Archived (abandoned)
