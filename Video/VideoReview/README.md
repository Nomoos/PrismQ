# VideoReview

**Workflow Stage: Video Production Phase - Quality Review**

## Overview

The **VideoReview** stage represents the quality assurance phase where assembled videos are reviewed for technical quality, content accuracy, and overall polish.

## Position in Workflow

```
VideoAssembly → [VideoReview] → VideoFinalized → Publishing
```

## Purpose

Final adjustments including captions, color grading, and smoothness verification. This stage ensures:

- Technical quality standards
- Content accuracy and completeness
- Professional polish
- Platform readiness
- Brand compliance

## Key Activities

1. **Technical Review** - Check video and audio quality
2. **Content Review** - Verify accuracy and completeness
3. **Caption/Subtitle QC** - Review and correct text overlays
4. **Color Grading** - Final color and look refinement
5. **Polish Pass** - Fix minor issues and enhance smoothness
6. **Platform Testing** - Verify compatibility and appearance

## Review Dimensions

### Technical Quality

**Video**
- [ ] Correct resolution and aspect ratio
- [ ] Proper frame rate (no dropped frames)
- [ ] No compression artifacts
- [ ] Smooth playback
- [ ] No glitches or errors
- [ ] Proper color space

**Audio**
- [ ] Clear voiceover (no distortion)
- [ ] Proper loudness level (-14 LUFS)
- [ ] No clipping or peaking
- [ ] Balanced music/voice mix
- [ ] No audio sync issues
- [ ] Clean transitions (no pops/clicks)

**Encoding**
- [ ] Correct codec and format
- [ ] Appropriate bitrate
- [ ] File size reasonable
- [ ] Metadata embedded
- [ ] Compatible with target platforms

### Content Quality

**Accuracy**
- [ ] All scripted content included
- [ ] No factual errors
- [ ] Correct spelling and grammar
- [ ] Proper attribution/credits
- [ ] Brand guidelines followed

**Completeness**
- [ ] Opening/hook present
- [ ] All main sections included
- [ ] Call-to-action included
- [ ] End card/outro present
- [ ] Required disclaimers added

**Engagement**
- [ ] Pacing maintains interest
- [ ] Visuals support message
- [ ] Clear narrative flow
- [ ] Strong opening and closing
- [ ] Appropriate length

### Visual Polish

**Color Grading**
- [ ] Consistent color throughout
- [ ] Proper exposure (not too dark/bright)
- [ ] Appropriate contrast
- [ ] Color mood matches intent
- [ ] No color shifts/jumps

**Graphics & Text**
- [ ] All text readable at target size
- [ ] No typos or formatting errors
- [ ] Animations smooth
- [ ] Graphics properly aligned
- [ ] Brand elements correct

**Transitions**
- [ ] Smooth and appropriate
- [ ] Not distracting
- [ ] Consistent style
- [ ] Proper timing

### Captions & Subtitles

**Accuracy**
- [ ] All spoken words included
- [ ] Correct spelling and punctuation
- [ ] Proper speaker identification (if multi-voice)
- [ ] Sound effects noted [if needed]

**Timing**
- [ ] Sync with audio perfectly
- [ ] Comfortable reading pace
- [ ] Proper line breaks
- [ ] Appropriate display duration

**Formatting**
- [ ] Readable font and size
- [ ] Good contrast with background
- [ ] Proper positioning
- [ ] Consistent style throughout

## Review Process

### First Review Pass
1. **Full Playthrough** - Watch complete video without stopping
2. **Take Notes** - Document issues as they appear
3. **Prioritize Issues** - Categorize (critical, important, minor)
4. **Create Feedback** - Compile detailed revision notes

### Second Review Pass
1. **Technical Check** - Focus on quality metrics
2. **Frame-by-Frame** - Review key moments in detail
3. **Audio Analysis** - Check waveforms and levels
4. **Export Verification** - Test different platforms/devices

### Revision Cycle
1. **Send Feedback** - Provide clear, actionable notes
2. **Implement Fixes** - Address all identified issues
3. **Re-Review** - Verify corrections made
4. **Iterate** - Repeat until standards met

## Revision Types

### Critical (Must Fix)
- Technical errors (glitches, corruption)
- Audio sync issues
- Major factual errors
- Brand guideline violations
- Platform incompatibility

### Important (Should Fix)
- Minor quality issues
- Small pacing problems
- Text typos
- Color inconsistencies
- Minor audio balance issues

### Minor (Nice to Fix)
- Subtle polish improvements
- Minor timing adjustments
- Optional enhancements
- Style preferences

## Deliverables

- Review feedback document
- Annotated timeline (if applicable)
- Comparison screenshots (before/after)
- Technical quality report
- Approval status decision

## Quality Gates

Video must meet ALL these criteria to pass:
- ✅ No critical technical issues
- ✅ Content accurate and complete
- ✅ Professional visual quality
- ✅ Clean audio mix
- ✅ Captions accurate and timed
- ✅ Platform compatibility verified
- ✅ Brand guidelines followed
- ✅ Target length achieved

## Transition Criteria

Video moves to VideoFinalized when:
- ✅ All review feedback addressed
- ✅ All quality gates passed
- ✅ Final approval received
- ✅ Platform testing complete
- ✅ No outstanding issues
- ✅ Ready for final export

## Platform-Specific Checks

### YouTube
- [ ] 16:9 aspect ratio
- [ ] Thumbnail optimized
- [ ] End cards functional
- [ ] Safe zones respected
- [ ] Metadata ready

### TikTok/Reels
- [ ] 9:16 vertical format
- [ ] Opening hook in first 3 seconds
- [ ] Text readable on mobile
- [ ] Under 60 seconds (or target length)
- [ ] Trending audio (if applicable)

### Instagram Feed
- [ ] 1:1 square format
- [ ] Under 60 seconds
- [ ] Mobile-optimized viewing
- [ ] Bold visuals for thumb-stopping
