# PrismQ Storytelling Elements Guide

**Comprehensive Guide to Story Structure and Narrative Elements**

This guide is available in the repository at `_meta/docs/STORYTELLING_GUIDE.md`.

Due to its comprehensive nature (22KB+), it has been created as a detailed reference document covering:

## Contents

1. **Story Foundation** (Základy příběhu)
   - Idea (Nápad) - Basic spark
   - Premise (Premisa) - 1-3 sentence explanation
   - Logline - One-sentence WOW effect
   - Hook (Háček) - Opening moment

2. **Story Structure** (Struktura příběhu)
   - Skeleton (Kostra) - 3-6 point overview
   - Outline (Osnova) - Detailed plan
   - Beat Sheet (Taktová mapa) - Small moments
   - Scenes (Scény) - Building blocks

3. **Narrative Elements** (Narativní prvky)
   - POV (Úhel pohledu) - Perspective
   - Emotional Arc (Emoční oblouk) - Character journey
   - Reveal (Odhalení) - Key information moments
   - Twist (Zvrat) - Unexpected changes
   - Climax (Vrchol) - Peak moment
   - Ending (Závěr) - Three types: open/shock/emotional

## Quick Reference

### For Short Content (TikTok/Reels - 60 seconds):
```
Hook (3s) → Setup (10s) → Rising Action (20s) → Twist (15s) → Ending (12s)
```

### Story Foundation Elements:

**Idea:** Single spark (word/sentence/feeling)
**Premise:** 1-3 sentences explaining what/why
**Logline:** One dramatic sentence for WOW
**Hook:** First moment that grabs attention

### Ending Types:

1. **Open Ending** - Leaves questions, perfect for creepy stories
2. **Shock Ending** - One final sentence hits hard  
3. **Emotional Ending** - Melancholy, reflection, realization
