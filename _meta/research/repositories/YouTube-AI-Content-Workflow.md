## Overview

Automated workflow for YouTube video summarization, blog generation, and podcast creation using open-source tools and n8n

## Key Features

- **Video Summarization** - Extract key points from YouTube videos
- **Blog Generation** - Create blog posts from video content
- **Podcast Creation** - Generate podcast episodes
- **n8n Integration** - Workflow automation
- **Open-source Tools** - Uses freely available tools

## Technology Stack

- **Primary Language:** Python
- **Workflow Platform:** n8n
- **Pipeline:** Video → Summary → Blog/Podcast

## Comparison to PrismQ

### Strengths
- ✅ **Multi-format output** - Blog and podcast from single source
- ✅ **YouTube integration** - Video platform focus
- ✅ **Workflow automation** - n8n-based orchestration

### Limitations
- ❌ **Reverse workflow** - Video → Text/Audio (opposite of PrismQ)
- ❌ **Depends on n8n** - External platform dependency
- ❌ **Content repurposing** - Not original content creation

## Relevance to PrismQ

**Medium relevance** - Interesting reverse workflow (video → text), but PrismQ focuses on creation, not repurposing.

## Extractable Mechanics

1. **Content Repurposing** - Techniques for transforming existing content
2. **YouTube API Integration** - Video platform automation
3. **Multi-format Export** - Single input, multiple outputs

## Implementation Priority

**Low** - Reverse workflow is not aligned with PrismQ's direction, but YouTube API patterns may be useful.
