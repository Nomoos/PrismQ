## Overview

Automates creation of Reddit narration videos with AI voiceovers and subtitles

## Key Features

- **Reddit Scraping** - Automated content sourcing
- **AI Voiceovers** - Eleven Labs TTS integration
- **Subtitle Generation** - OpenAI Whisper for timing
- **Video Editing** - MoviePy for assembly
- **Complete Automation** - End-to-end pipeline

## Technology Stack

- **Primary Language:** Python
- **TTS:** Eleven Labs
- **Subtitles:** OpenAI Whisper
- **Video:** MoviePy
- **Source:** Reddit API
