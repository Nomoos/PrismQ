# Issue Tracking

This directory contains issue tracking and project management files for PrismQ.IdeaInspiration.Classification.

## Directory Structure

- **new/** - New issues that haven't been started yet
- **wip/** - Work in progress issues currently being addressed
- **done/** - Completed issues

## Files

- **KNOWN_ISSUES.md** - List of known issues and limitations
- **ROADMAP.md** - Future development plans and feature roadmap

## Using This Directory

### Creating New Issues

1. Create a new markdown file in `new/` directory
2. Use format: `YYYY-MM-DD-issue-name.md`
3. Include clear description and acceptance criteria
4. Move to `wip/` when starting work
5. Move to `done/` when completed

### Workflow

```
new/ → wip/ → done/
```

## Issue Template

```markdown
# Issue Title

## Description
Clear description of the issue or feature request.

## Current Behavior
What happens now.

## Expected Behavior
What should happen.

## Acceptance Criteria
- [ ] Criterion 1
- [ ] Criterion 2

## Notes
Additional context or information.
```
