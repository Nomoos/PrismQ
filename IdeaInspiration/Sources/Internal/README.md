# Internal Sources

Internally managed content ideas and data imports.

## Overview

Internal sources represent manually curated and imported data that doesn't come from external platforms.

## Sources

- **ManualBacklogSource**: Manually curated idea backlog
- **CSVImportSource**: Bulk data imports from CSV files

## Purpose

Internal sources help:
- Manage curated idea lists
- Import historical data
- Track manual research
- Organize custom datasets
- Maintain internal knowledge
