# Work In Progress

Place issues you are currently working on here.

## Workflow

1. Move issue from `../new/` when you start working on it
2. Keep the issue file updated with progress notes
3. Move to `../done/` when completed

## Progress Notes

Add progress notes to the issue file as you work:

```markdown
## Progress

### 2025-01-15
- Started implementation
- Created initial structure

### 2025-01-16
- Completed feature X
- Added tests
```
