# New Issues

Place new issues here as markdown files.

## Format

Use the following format for issue files:

```
YYYY-MM-DD-short-description.md
```

Example: `2025-01-15-add-youtube-source.md`

## Template

```markdown
# Issue: [Title]

**Created**: YYYY-MM-DD
**Priority**: High/Medium/Low
**Category**: Bug/Feature/Enhancement/Documentation

## Description
Clear description of the issue.

## Acceptance Criteria
- [ ] Criterion 1
- [ ] Criterion 2
- [ ] Criterion 3

## Notes
Additional notes or context.
```

When you start working on an issue, move it to `../wip/`.
