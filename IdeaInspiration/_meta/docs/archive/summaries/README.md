## Purpose

These documents are kept for historical reference and provide insight into:
- Project milestones and completion criteria
- Evaluation methodologies and results
- Module registration processes
- Phase completion verification

## When to Archive

Move documents to this directory when:
- A project phase is completed
- An evaluation or assessment is finished
- A major milestone is reached and documented
- Summary documents are no longer actively maintained
